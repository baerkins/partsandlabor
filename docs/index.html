    grid-zed-reset

    @mixin grid-zed-reset($nth: 1, $gutter-value: $target-gutter) { ... }

    Description

    Remove the effects of a grid-zed include.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $nth

    Target nth-child

    		String1
    $gutter-value

    If you are using a different gutter than default

    		String$target-gutter

    Example

    Basic Usage Sass

    /* Span 6 columns than 4 columns of 12 column grid */
.post {

  @include break($small) {
    @include grid-col(6);
    @include grid-zed(2n);
  }

  @include break($medium) {
    @include grid-zed-reset(2n);
    @include grid-col(4);
    @include grid-zed(3n);
  }
}

    Requires

    • [function] calc-gutter-width
    • [variable] calc

    grid-zed

    @mixin grid-zed($nth: 1) { ... }

    Description

    Remove margin-right from target element and clear floats on the following element of that type. Can be used to target the last item in a row via nth-child. Modeled after Bourbon Neats omega

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $nth

    Target nth-child

    		String1

    Example

    Basic Usage Sass

    .no-margin {
  @include grid-col(8);
  @include grid-zed();
}

/* Span 3 columns of 12 column grid */
.three-col {
  @include grid-col(4);
  @include grid-zed(3n);
}

    flex-grid-zed-reset

    @mixin flex-grid-zed-reset($nth: 1) { ... }

    Description

    Break flex items to a new line. Not always neccessary, but useful in some circumstances, such as a row of two column elements in a twelve column grid that should break after six elements.

  /* Span 2 columns in 12 column grid */

    @include flex-grid-col(2);
    @include flex-grid-zed(6n);
  }

  /* Span 3 columns in 12 column grid */

    @include flex-grid-zed-reset(6n);
    @include flex-grid-col(3);
  }
}

    Description

    Create a new breakpoint. Use in conjunction with breakpoint variables.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    Breakpoint size

    		String— none
    $attr

    Media break attribute (min-width or max-width)

    		Stringmin-width
    $media

    Media type

    		Stringonly screen

    Content

    This mixin allows extra content to be passed (through the @content directive).
