Parts and Labor

Parts and Labor is a sass library of utilities and grid systems. This library does not output any css. It is purely a library of utilities and mixins for you to use in your sass projects.

Install Parts and Labor

Install via npm:

npm install --save-dev partsandlabor

Import the base file in your sass file:

@import '../node_modules/partsandlabor/partsandlabor';

To overwrite default variables, import a sass file with variable names before importing Parts and Labor:

@import '../utils/partsandlabor_overwrites';
@import '../node_modules/partsandlabor/partsandlabor';
